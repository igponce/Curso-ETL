# Curso-ETL
Curso de ETL

Este curso está en formato [mdBook](https://github.com/rust-lang/mdBook) para poder acceder y buscar fácilmente el contenido.

Para compilar el contenido necesitas instalar mdbook en el directiorio /bin
La forma más sencilla de instalarlo es haciendo que cargo lo instale directamente del git.

```bash
$ cd bin
$ cargo install --git https://github.com/rust-lang/mdBook.git mdbook
```

Una vez instalado mdbook, ejecuta `bin/mdbook serve`y mdbook compilará en html el contenido.
Puedes acceder al libro en [http://localhost:3000/](http://localhost:3000/)
