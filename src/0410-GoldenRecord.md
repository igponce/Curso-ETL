# Golden Record

Un "Golden Record" (Registro Dorado) es el registro único, completo, exacto y verificable de un objeto o individuo, que resulta de la integración de todos los datos disponibles sobre él provenientes de distintas fuentes. 

Ejemplos de Goldn Record:

- Ficha maestra de clientes
- Ficha maestra de productos
- Ficha maestra de contratos...

Según DAMA DMBoK, es el resultado final de un proceso de integración y calidad de datos donde se consolidan múltiples instancias de datos del mismo objeto en una sola versión "verdadera" y confiable que puede ser utilizada en toda la organización.

No es necesario disponer de un Golden Record para poder cumplir con normativas de protección de datos, pero es muy conveniente.
