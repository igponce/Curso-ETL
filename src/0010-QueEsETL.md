# Qué es una ETL

Una ETL consiste en un:

- Movimiento de datos
- Transformación
- Carga en otro sistema

Esto hay que hacerlo de forma **Ordenada**, **Controlada**, y **periódicamente**.

- No es una operación "one-shot".
- Es un proceso que se repetirá en el tiempo.
- Esto implica que la organización tiene que poner recursos materiales y humanos para llevarlo a cabo.
- Para que esto ocurra debemos que poner en valor el beneficio de hacer este proceso.
